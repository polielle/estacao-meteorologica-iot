# 🌦 Estação Meteorológica IoT

Projeto acadêmico desenvolvido para monitoramento climático em tempo real utilizando ESP32.

## 🚀 Tecnologias Utilizadas

- ESP32
- DHT22
- BMP280
- LDR
- InfluxDB
- Grafana
- Arduino IDE

## 📊 Funcionalidades

- Coleta de temperatura
- Monitoramento de umidade
- Medição de pressão atmosférica
- Monitoramento de luminosidade
- Envio de dados para InfluxDB
- Dashboard em tempo real no Grafana

## 🖥 Dashboard

O sistema utiliza Grafana para visualização dos dados climáticos em tempo real.

## 📚 Objetivo

Desenvolver uma estação meteorológica de baixo custo utilizando tecnologias IoT e armazenamento em nuvem.
